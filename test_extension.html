Chrome Extension Test Page

This is a simple test page to verify that the text token colorizer extension works correctly.

The extension should:

  • Extract all text from this page
  • Tokenize the text using the GPT-OSS-20B tokenizer (or fallback)
  • Colorize each token based on its token ID
  • Show logarithmic token ID as superscript
  • Use blue for low token IDs and red for high token IDs

Here's some sample text with various complexities:

Simple text: Hello world!

Complex text: The quick brown fox jumps over the lazy dog.

Technical text: Machine learning algorithms use transformers for natural language processing.

Punctuation: What's up? I'm doing great! How about you?

If the extension is working correctly, you should see each word and punctuation mark colored differently based on its token ID, with small superscript numbers showing the logarithmic token ID values.
